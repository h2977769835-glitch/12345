当前工程文件夹内包含了腾讯广告算法大赛官方所发布的SASRec（Self-Attentive Sequential Recommendation）基准模型源码。
在多维特征提取与处理环节，为了充分挖掘并利用赛题中提供的丰富多模态信息，我特别调用了核心模块 model.py 中定义的 save_item_emb 函数（位于第401行）。
该方法通过特定的特征融合机制，将异构的多模态数据进行对齐与联合编码，从而计算并导出富含语义的物品嵌入表示（item embedding）。
这一步骤产出的高质量向量有效表征了物品的综合特征，并作为关键的底层数据输入，直接服务于后续码本（codebook）的生成与聚类构建流程。
